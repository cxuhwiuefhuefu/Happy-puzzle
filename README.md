# 技术栈
html + css + js + jQuery
# 流程思路
分为展示层 业务层 数据层
<Br/>
展示层用html和css来展示
<Br/>
业务层先初始化Dom和初始化标准数组 点击开始按钮图片重排和数组乱序 拖拽图片会图片重排和交换数组紧接着有个交换判定 最终完成判定游戏成功 游戏又变为开始的状  态 图片重排
<Br/>
数据层由标准数组和动态数组组成
